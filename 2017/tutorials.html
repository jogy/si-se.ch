Hands-On: Machine Learning for Engineers (1. Teil)

Dr. Roman Bertolami, Principal Consultant, Zühlke
Florian Lüscher, Software Engineer, Zühlke

Die Liste der erfolgreichen Anwendungsfälle von Machine Learning wird immer länger. Softwareentwickler und IT-Architekten müssen sich mit den Konzepten von Machine Learning vertraut machen und sich mit dem Engineering von entsprechenden Anwendungen auseinandersetzen. Im Tutorial bieten wir einen Einstieg ins Thema indem wir Hands-On ein neuronales Netz trainieren, um eine Beispielaufgabe zu lösen. Dazu verwenden wir das Framework TensorFlow. Das Tutorial richtet sich an Teilnehmer mit technischem Hintergrund. Vorwissen im Gebiet ist nicht nötig, jedoch sollten Teilnehmer einen Laptop mitbringen, der in der Lage ist Docker-Container auszuführen.

  • Einführung in Machine Learning
  • Übersicht von verschiedenen Methoden für Machine Learning
    • Supervised vs. Unsupervised Learning
    • Classification
    • Regression
  • Hands-On Entwickeln eigener Modelle
    • Einführung in Tensorflow
    • Lineare Regression
    • Neuronale Netze
    • Qualitätsmessung der trainierten Modelle
  • Ausblick
